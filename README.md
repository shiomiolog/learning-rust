# learning-rust 🦀

Rust言語の基礎学習記録をまとめるリポジトリです。

## 🗺️ 学習コンテンツ

- [ ] **[The Rust Programming Language (The Book)](https://doc.rust-jp.rs/book-ja/)**
  - 公式ドキュメントの写経や、各章の動作確認コード。
- [ ] **[Rustlings](https://github.com/rust-lang/rustlings)**
  - 公式の演習問題集の解答記録。

## 📂 ディレクトリ構成

- `the-book/` : The Bookを読みながら書いた検証用コードやメモ
- `rustlings/` : Rustlingsのローカルリポジトリ（または解答のみを配置）
- `sandbox/` : その他、ちょっとした文法のテストや実験用のコード

## 🛠️ 開発環境

- Rust (cargo, rustc)
